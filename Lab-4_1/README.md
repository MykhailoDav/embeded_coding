# Lab 4.1 - Intermediate AVR Project

## Description
Fourth laboratory assignment focusing on intermediate AVR programming concepts and peripheral interfacing.

## Hardware Requirements
- ATmega328P microcontroller
- Additional components (see source code)

## Building and Uploading

```bash
cd Lab-4_1
pio run -t upload
```

- Real-world embedded applications
